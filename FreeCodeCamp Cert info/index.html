Table

Tables are for structured, tabular data — not for page layout.

Property Value What it does
display: flex Flexbox One-axis layout (row or column)
display: grid Grid Two-axis layout (rows and columns)
position: relative Relative Offset from its normal position
position: absolute Absolute Placed relative to nearest positioned ancestor
position: fixed Fixed Stays in place when you scroll
position: sticky Sticky Scrolls normally until it hits a threshold, then sticks

Form Elements

Forms collect user input. Each <input> should have a <label> — the for attribute matches the input's id so clicking the label focuses the input.

Images

The alt attribute describes the image for screen readers and displays when the image fails to load. Always include it.
